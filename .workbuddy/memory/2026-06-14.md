## 2026-06-14 项目打磨 — MD 公式格式修正

### 诊断
- lesson-01（Qoder 原版）公式用独立 `$$` 块 + `\xrightarrow{点燃}` 格式 ✅
- lesson-02~10（AI 再生版）公式用内联 `$...$` + `\xrightarrow{\text{点燃}}` 格式 ❌
- 根因：KaTeX 组件未加载 `\text` 扩展，`\text{中文}` 会渲染失败
- 根因2：convert-md.mjs 只识别 `$$` 独立块为 equation，内联 `$...$` 被当普通文本

### 写入脚本
- `scripts/fix-md-format.mjs` — 自动修正 MD 公式格式，零 token 消耗
  - 规则1：`\text{中文}` → `中文`（去除 KaTeX 不支持的 \text 包裹）
  - 规则2：`> callout` 中公式 → 提取完整方程式到独立 `$$` 块
  - 规则3：方程式行（前后中文<=5）→ 内联转独立 `$$`
  - 规则4：引用句（前后中文>5）→ 保持内联
  - 保留 `\text{ g}`、`\text{ kg}` 等单位文本

### 修正结果
- 修正 6 个文件：lesson-02, 03, 07, 08, 09, 10
- lesson-02 → 7 个公式块，callout/list 公式全部正确提取
- lesson-03 → `\text{MnO₂}` → `MnO₂` ✅
- lesson-08 → 21 个公式块，句子引用保持内联，`\text{ g}` 保留
- 删除冗余文件 `lesson-02-化学是一门以实验为基础的科学.md`
- `npm run convert` + `npm run build` 均成功
