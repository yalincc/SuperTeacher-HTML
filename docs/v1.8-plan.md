# v1.8 规划：多学科支持 + 首页重构

> 日期：2026-06-18 | 状态：讨论中（未定稿）

---

## 背景

当前架构只支持单一课程（初中化学），所有课时共用一套 ID 编号。新增物理课程时，物理 lesson-11 混入了化学编号体系。首页只展示一个学科，无法切换。

需要改造为多学科架构，每个学科独立课程配置和课时数据。

---

## 目标

1. 支持多学科（化学、物理等），每个学科独立课程配置和课时数据
2. 首页重构为学科选择入口 + 各学科课程列表
3. 清理物理 lesson-11 混入化学编号的问题

---

## 数据架构改造

### 目录结构

```
src/data/
├── courses/
│   ├── chemistry/
│   │   ├── course.json          ← 化学课程配置
│   │   ├── lesson-01.json       ← 化学课时（独立编号 1-10）
│   │   ├── ...
│   │   └── lesson-10-exercises.json
│   └── physics/
│       ├── course.json          ← 物理课程配置
│       ├── lesson-01.json       ← 物理课时（独立编号从 1 开始）
│       └── lesson-01-exercises.json
├── index.ts                     ← 聚合导出所有课程
```

### course.json 格式（每个学科一份）

```json
{
  "id": "chemistry",
  "course": { "name": "初中化学", "icon": "⚗️", "subtitle": "人教版九年级全一册", "color": "#e8600c" },
  "schedule": { "weeks": 5, "weekLabels": [...], "weekTitles": [...] },
  "lessons": [ { "id": 1, "title": "走进化学世界", ... } ],
  "features": { "katex": true, "formulaNormalize": true }
}
```

## 路由改造

### 当前路由

```
/                       → HomePage（单课程）
/lesson/:id             → LessonPage
/lesson/:id/game        → GamePage
```

### 新路由

```
/                       → HomePage（学科选择卡片页）
/course/:courseId        → CoursePage（某学科课程页面）
/course/:courseId/lesson/:id      → LessonPage
/course/:courseId/lesson/:id/game → GamePage
```

## 扩展性：如何加新学科

加新学科**只需一步**：在 `src/data/courses/` 下新建目录，放入文件即可。

```
src/data/courses/biology/          ← 新建目录
├── course.json                    ← 课程配置（id: "biology"）
├── lesson-01.json                 ← 课时数据
└── lesson-01-exercises.json       ← 练习题
```

首页自动出现生物学科卡片，无需改任何代码。

**原理**：`src/data/index.ts` 使用 Vite 的 `import.meta.glob` 自动扫描 `courses/*/course.json` 和 `courses/*/lesson-*.json`，无需手动 import。

**唯一约束**：
- 目录名即 courseId（小写英文）
- 必须有 `course.json`
- 课时文件命名：`lesson-{数字}.json`、`lesson-{数字}-exercises.json`
