# v1.6 架构重构方案：知识点 + 练习分离

> 状态：**✅ 已实施** | 日期：2026-06-16

---

## 当前问题

当前 `LessonPage` 把知识点和练习题放在同一页面，游戏元素（心数、GIF 动画）硬加在练习区，体验不协调。

---

## 新架构：两个独立路由

```
/lesson/:id          → 知识点页面（普通网页风格）
/lesson/:id/game     → 练习游戏页面（互动游戏风格）
```

### 知识点页面 `/lesson/:id`

- 纯展示型网页，注重阅读体验
- 学习目标 + 知识点 + 典型例题
- 无游戏元素，无心数，无动画
- 底部有「开始挑战」按钮 → 跳转到游戏页面
- 风格：干净、专业、类似教材

### 练习游戏页面 `/lesson/:id/game`

- 互动游戏风格，注重参与感
- 课后练习（选择/判断/填空/简答）
- 心数系统 + GIF 动画
- 答完显示本课小结
- 底部有「重做题目」和「返回知识点」按钮
- 风格：活泼、有趣、类似游戏

## 文件改动

### 新增

| 文件 | 说明 |

| `src/pages/GamePage.tsx` | 练习游戏页面（从 LessonPage 拆出） |

| `src/components/game/GameFooter.tsx` | 游戏页底部（重做 + 返回） |

### 改造

| 文件 | 改动 |
|------|------|
| `src/App.tsx` | 新增 `/lesson/:id/game` 路由 |
| `src/pages/LessonPage.tsx` | 移除游戏元素，只保留知识点 + 「开始挑战」按钮 |
| `src/pages/HomePage.tsx` | 课时卡片点击进入知识点页面 |

### 保留不变

| 文件 | 说明 |
|------|------|
| `src/hooks/useGame.ts` | GameContext 不变 |
| `src/components/exercises/*` | 练习题组件不变 |

## 用户流程

```
首页
  ↓ 点击课时
知识点页面（阅读学习）
  ↓ 点击「开始挑战」
游戏页面（答题互动）
  ↓ 答完所有题
显示本课小结
  ↓ 点击「返回知识点」或「下一课」
知识点页面 / 下一课
```
