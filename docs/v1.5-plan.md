## 架构变更：Markdown → JSON

### 当前架构（有解析层）

```
docs/curriculum/*.md → scripts/convert-md.mjs → src/data/lessons/*.json → React
```

### 新架构（AI 直接生成 JSON）

```
AI 直接输出 JSON → src/data/lessons/*.json → React
```

**优势**：
- 少一层解析 = 少 N 多 bug
- AI 生成 JSON 稳定性远高于 MD 解析
- JSON 格式严格，不易出歧义

### 迁移计划

| 步骤 | 说明 |

## 技术方案

### 新增内容块类型：AnimationBlock

```typescript
// types/index.ts 新增
export interface AnimationBlock {
  type: 'animation'
  src: string        // GIF 文件路径
  alt?: string       // 替代文字
  width?: number     // 宽度（可选）
}
```

### JSON 中的使用方式

```json
{
  "type": "animation",
  "src": "/gifs/firework.gif",
  "alt": "庆祝烟花"
}
```

### 过关系统架构

```
GameContext（新增）
├── hearts: number          // 当前心数
├── unlockedModules: Map    // 每课解锁的模块
├── useHeart()             // 扣心/重置
├── unlockModule()         // 解锁模块
└── isModuleUnlocked()     // 检查是否解锁
```
