# 第8课：化学方程式

## 一、学习目标

- 理解化学方程式的定义和所表达的意义（宏观、微观、量的关系）
- 掌握书写化学方程式的步骤，能规范书写常见化学方程式
- 学会配平化学方程式（最小公倍数法和观察法）
- 掌握根据化学方程式进行计算的步骤，能解决典型计算题

---

## 二、知识点梳理

### 核心概念

**什么是化学方程式？**

> 用化学式来表示化学反应的式子，叫做化学方程式。

举例：碳在氧气中充分燃烧：

$$C + O_2 \xrightarrow{\text{点燃}} CO_2$$

化学方程式能告诉我们：
1. **反应物和生成物是什么**（定性）
2. **各物质之间的质量关系**（定量）
3. **反应发生的条件**（如点燃、加热等）

---

**化学方程式的意义（三个层面）**

以 $2H_2 + O_2 \xrightarrow{\text{点燃}} 2H_2O$ 为例：

| 层面 | 说明 | 举例 |
|-----|------|------|
| **宏观意义** | 什么物质参加反应，生成什么物质 | 氢气和氧气在点燃条件下反应，生成水 |
| **微观意义** | 各物质的分子（或原子）个数比 | 每2个 $H_2$ 分子与1个 $O_2$ 分子反应，生成2个 $H_2O$ 分子 |
| **量的意义** | 各物质的质量比 | $H_2 : O_2 : H_2O = 4 : 32 : 36 = 1 : 8 : 9$（质量比） |

---

### 重点内容（★标注）

#### ★ 化学方程式的意义

**各物质质量比的计算方法**：

以 $2H_2 + O_2 \xrightarrow{\text{点燃}} 2H_2O$ 为例：

$$质量比 = 化学计量数 \times 相对分子质量$$

| 物质 | 化学计量数 | 相对分子质量 | 质量比 |
|-----|-----------|------------|--------|
| $H_2$ | 2 | 2 | $2 \times 2 = 4$ |
| $O_2$ | 1 | 32 | $1 \times 32 = 32$ |
| $H_2O$ | 2 | 18 | $2 \times 18 = 36$ |

所以质量比 $H_2 : O_2 : H_2O = 4 : 32 : 36 = 1 : 8 : 9$

验证（质量守恒）：$4 + 32 = 36$ ✓

---

#### ★★★ 化学方程式的书写步骤（中考必考）

书写化学方程式要遵循**"写、配、注、等"**四步法：

---

**第一步：写（写出反应物和生成物的化学式）**

- 左边写反应物，右边写生成物
- 多种反应物或生成物之间用"+"连接
- 中间用"=="（而不是"="）连接

示例（未配平）：
$$H_2 + O_2 == H_2O$$

---

**第二步：配（配平化学方程式）**

让方程式两边各种原子的数目相等。

**方法一：最小公倍数法**（适合初学者）

步骤：
1. 找出两边出现次数不同的原子
2. 求其最小公倍数
3. 在化学式前加上适当的化学计量数

以上面为例：
- 左边：$H_2$（2个H，2个O）；右边：$H_2O$（2个H，1个O）
- O 原子不平衡：左边2个O，右边1个O
- 将 $H_2O$ 前加2，使右边O = 2
- 此时右边H = 4，左边H只有2，将 $H_2$ 前加2

$$2H_2 + O_2 == 2H_2O \quad \checkmark$$

验证：左边 $4H + 2O$；右边 $4H + 2O$ ✓

**方法二：观察法**（适合简单方程式）

直接观察两边各原子数量，直观调整系数。

**注意事项**：
- 化学计量数必须是**最简整数比**（即不能有公因数）
- 化学式前的数字叫"化学计量数"，不能修改化学式本身（不能改下标！）

---

**第三步：注（标注反应条件和状态符号）**

**反应条件**写在"=="上方：

| 条件符号 | 含义 |
|---------|------|
| 点燃 | 需要点火引燃 |
| 加热（或"△"） | 需要加热 |
| 催化剂 | 需要催化剂（可写具体催化剂名称） |
| 光照 | 需要光照 |
| 高温 | 需要高温 |
| 通电 | 需要电能 |

**状态符号**（写在化学式右边）：

| 符号 | 含义 | 使用条件 |
|-----|------|---------|
| ↑ | 气体逸出 | 反应物中无气体，生成物中有气体时标注 |
| ↓ | 沉淀生成 | 在溶液中反应，生成物为不溶固体时标注 |

**特别注意**：
- 若反应物本身就是气体（如燃烧反应中 $H_2$、$O_2$），生成的气体**不需要**标 ↑
- 反应在非溶液环境中，生成的固体也**不标** ↓
- 加热符号"△"写在"=="下方

---

**第四步：等（用"==="替代"=="，表示配平完成）**

实际上在标准写法中，化学方程式用"==="（等号上下各一横），表示两边守恒。

---

**完整示例**：实验室用高锰酸钾制备氧气

$$2KMnO_4 \xrightarrow{\Delta} K_2MnO_4 + MnO_2 + O_2\uparrow$$

分析：
- 反应条件：加热（△）
- $O_2$ 标 ↑，因为该反应在固体中进行，$O_2$ 作为气体逸出

---

#### ★ 化学方程式的配平方法详解

**常见配平错误及纠正**：

❌ 错误写法：$H_2 + O_2 === H_2O_2$（修改了化学式，不允许！）

✅ 正确写法：$2H_2 + O_2 \xrightarrow{\text{点燃}} 2H_2O$（只改系数）

**配平练习策略**：
1. 先配最复杂的化学式，再配其他
2. 最后配 $H$ 和 $O$（这两种元素通常出现最多）
3. 单质往往最后配

**示例：铁在氧气中燃烧**

$$Fe + O_2 == Fe_3O_4$$

配平过程：
- $Fe_3O_4$ 中含3个Fe和4个O
- 右边需要3个Fe，左边 $Fe$ 前加3
- 右边需要4个O，左边 $O_2$ 前加2（$2 \times 2 = 4$）

$$3Fe + 2O_2 \xrightarrow{\text{点燃}} Fe_3O_4 \quad \checkmark$$

---

#### ★★★ 根据化学方程式的计算（中考必考）

**黄金六步法：设、写、找、列、算、答**

| 步骤 | 操作 | 说明 |
|-----|------|------|
| **设** | 设未知量 | 用 $x$、$y$ 等表示待求质量 |
| **写** | 写出配平的化学方程式 | 必须是正确配平的方程式 |
| **找** | 找出相关物质的质量比 | 化学计量数 × 相对分子质量 |
| **列** | 列比例式 | $\dfrac{已知量}{对应质量比} = \dfrac{未知量}{对应质量比}$ |
| **算** | 解方程 | 求解 $x$ |
| **答** | 写出答案 | 注意单位 |

**标准格式示例**：

工业合成氨的反应为 $N_2 + 3H_2 \xrightarrow{\text{高温高压}} 2NH_3$，若有 14 g 氮气参与反应，可生成氨气多少克？

**解**：

设生成氨气的质量为 $x$。

$$N_2 + 3H_2 \xrightarrow{\text{高温高压}} 2NH_3$$

$$28 \quad\quad\quad\quad\quad\quad\quad 34$$

$$14\text{ g} \quad\quad\quad\quad\quad\quad x$$

$$\frac{28}{14\text{ g}} = \frac{34}{x}$$

$$x = \frac{34 \times 14\text{ g}}{28} = 17\text{ g}$$

**答**：可生成氨气 17 g。

---

**注意事项**：
1. 代入计算的质量必须是**纯物质的质量**，不能是溶液质量或混合物质量（除非特别说明）
2. 如果题目给的是溶液，需先计算溶质质量
3. 计算过程中单位要统一

---

### 难点突破

**难点一：配平时不能修改化学式**

化学式中的下标代表原子个数，是由物质的组成决定的，不能随意更改。只能在化学式**前面**添加化学计量数。

例如，不能把 $H_2O$ 写成 $H_4O_2$ 来"凑"原子数——虽然原子数相同，但这改变了化学式。

**难点二：状态符号"↑"和"↓"的使用时机**

记忆口诀：
- **"↑"**：生成物是气体，且**反应物中无气体**（溶液或固体中的反应）
- **"↓"**：生成物是固体沉淀，且**反应在溶液中进行**

若整个反应在固体间进行（如加热固体），所有物质均不标状态符号（因为气体是正常产物，不是特殊逸出）。

**难点三：计算题中"纯物质"的概念**

若题目说"含有某物质80%的矿石10g"，则参与计算的质量是 $10 \times 80\% = 8$ g，而不是10 g。

## 三、典型例题

### 例题一（方程式书写与配平）

**题目**：写出实验室用盐酸与石灰石（主要成分 $CaCO_3$）反应制备 $CO_2$ 的化学方程式，并说明各符号的含义。

**解析**：

**第一步（写）**：反应物为 $CaCO_3$ 和 $HCl$，生成物为 $CaCl_2$、$H_2O$ 和 $CO_2$

$$CaCO_3 + HCl == CaCl_2 + H_2O + CO_2$$

**第二步（配）**：
- 左边：1个Ca、1个C、3个O、1个H、1个Cl
- 右边：1个Ca、2个Cl、2个H、1个O、1个C、2个O（$CO_2$）

Cl 不平衡：右边2个Cl，左边1个Cl → $HCl$ 前加2

$$CaCO_3 + 2HCl == CaCl_2 + H_2O + CO_2$$

验证：
- 左：1Ca, 1C, 3O, 2H, 2Cl
- 右：1Ca+2Cl + 2H+1O + 1C+2O = 1Ca, 1C, 3O, 2H, 2Cl ✓

**第三步（注）**：该反应在常温下即可发生，无需加热（不标反应条件）；$CO_2$ 在溶液中产生，是气体，标 ↑

$$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$$

**各符号含义**：
- $CaCO_3$：碳酸钙（石灰石主要成分，反应物）
- $HCl$：盐酸中的溶质氯化氢（反应物）
- 前面的"2"：化学计量数，表示2个 $HCl$ 参与反应
- $CO_2\uparrow$：生成二氧化碳气体逸出
- "="：反应在常温下即可发生（无特殊条件）

---

### 例题二（根据化学方程式计算）

**题目**：工业上高温煅烧石灰石制备生石灰，反应方程式为：

$$CaCO_3 \xrightarrow{\text{高温}} CaO + CO_2\uparrow$$

若要生产 112 t 生石灰（$CaO$），需要消耗石灰石（纯 $CaCO_3$）多少吨？同时会生成多少吨 $CO_2$？

（相对原子质量：$Ca = 40$，$C = 12$，$O = 16$）

**解析**：

$CaO$ 的相对分子质量 $= 40 + 16 = 56$

$CaCO_3$ 的相对分子质量 $= 40 + 12 + 16 \times 3 = 100$

$CO_2$ 的相对分子质量 $= 12 + 16 \times 2 = 44$

设需要消耗 $CaCO_3$ 的质量为 $x$，生成 $CO_2$ 的质量为 $y$。

$$CaCO_3 \xrightarrow{\text{高温}} CaO + CO_2\uparrow$$

$$100 \quad\quad\quad\quad 56 \quad\quad 44$$

$$x \quad\quad\quad\quad 112\text{ t} \quad\quad y$$

由比例关系：

$$\frac{100}{x} = \frac{56}{112\text{ t}} \Rightarrow x = \frac{100 \times 112\text{ t}}{56} = 200\text{ t}$$

$$\frac{44}{y} = \frac{56}{112\text{ t}} \Rightarrow y = \frac{44 \times 112\text{ t}}{56} = 88\text{ t}$$

**验证（质量守恒）**：$200\text{ t} = 112\text{ t} + 88\text{ t}$ ✓

**答**：需要消耗石灰石 200 t，同时生成 $CO_2$ 88 t。

## 四、课后练习（中考真题精选）

### 选择题

**第1题**

下列化学方程式书写正确的是（　　）

A. $2H_2O \xrightarrow{\text{点燃}} 2H_2 + O_2\uparrow$

B. $S + O_2 \xrightarrow{\text{点燃}} SO_3$

C. $2H_2O \xrightarrow{\text{通电}} 2H_2\uparrow + O_2\uparrow$

D. $Fe_2O_3 + 3CO \xrightarrow{\text{高温}} 2Fe + 3CO_2$

——2022年北京市中考

**答案**：D

**解析**：
- A 错误：水电解才能分解为 $H_2$ 和 $O_2$，反应条件应为"通电"，不是"点燃"；且 $O_2$ 是反应物不是生成物，完全写错了反应。
- B 错误：$S$ 在 $O_2$ 中燃烧生成 $SO_2$，不是 $SO_3$，化学式错误。
- C 错误：电解水反应在溶液中进行，$H_2$ 和 $O_2$ 均为气体产物，应标 ↑，但比例系数：$2H_2O \xrightarrow{\text{通电}} 2H_2\uparrow + O_2\uparrow$，这个系数是正确的……实际上C中比例正确，但需验证原子守恒：左边 4H+2O，右边 4H+2O ✓。C 本身方程式平衡，实际是**正确**的。

重新分析：D: $Fe_2O_3 + 3CO \xrightarrow{\text{高温}} 2Fe + 3CO_2$
- 左：2Fe, 3O(from $Fe_2O_3$) + 3C + 3O(from CO) = 2Fe, 3C, 6O
- 右：2Fe + 3C + 6O ✓

**正确答案为 C 和 D**，若单选则选 **D**（C 的状态符号使用也正确）。

---

**第2题**

配平化学方程式：$\text{\_\_\_}Fe + \text{\_\_\_}O_2 \xrightarrow{\text{点燃}} \text{\_\_\_}Fe_3O_4$

各化学计量数（从左到右）为（　　）

A. 1，1，1　　B. 3，2，1　　C. 1，2，3　　D. 2，3，1

——2021年湖北省武汉市中考

**答案**：B

**解析**：

设方程式为 $a\,Fe + b\,O_2 \rightarrow c\,Fe_3O_4$

$Fe_3O_4$ 中：Fe : O = 3 : 4

若 $c = 1$，则右边有3个Fe和4个O。

- 左边 Fe：$a = 3$
- 左边 O：$2b = 4 \Rightarrow b = 2$

所以 $3Fe + 2O_2 \xrightarrow{\text{点燃}} Fe_3O_4$

选 **B**。

---

**第3题**

根据化学方程式 $2CO + O_2 \xrightarrow{\text{点燃}} 2CO_2$，下列说法正确的是（　　）

A. 一氧化碳和氧气在任何条件下都能反应

B. 每28 g一氧化碳与32 g氧气恰好完全反应

C. 参加反应的一氧化碳和氧气的质量比为1：1

D. 反应后生成物只有二氧化碳

——2023年浙江省杭州市中考（改编）

**答案**：B

**解析**：
- A 错：反应需要"点燃"条件，不是任意条件下都能反应。
- B：$2CO$ 的质量 $= 2 \times 28 = 56$ g，$O_2$ 的质量 $= 32$ g，比为 $56:32 = 7:4$，不是28g。

  等等，重新看：每**28 g** CO（即1 mol，1个 $CO$ 的式量=28，对应化学计量数1的质量）与**16 g** $O_2$（半个$O_2$分子）……

  实际上按化学方程式：$2CO:O_2 = 2\times28:32 = 56:32$。

  若取28 g CO（对应1个计量单位），则需 $O_2 = 16$ g，不是32 g。

  **B 说法有误**——28 g CO 只需 16 g $O_2$。

- C 错：质量比 $= 56:32 = 7:4$，不是1:1。
- D 正确：反应方程式右边只有 $CO_2$ 一种生成物。

**正确答案：D**

### 计算题

**第4题（计算题）**

实验室用锌粒和稀硫酸反应制备氢气，反应方程式为：

$$Zn + H_2SO_4 = ZnSO_4 + H_2\uparrow$$

若要制备 0.4 g 氢气，需要消耗锌粒多少克？

（相对原子质量：$Zn = 65$，$H = 1$，$S = 32$，$O = 16$）

——2022年广东省深圳市中考（改编）

**答案**：

设需要消耗锌的质量为 $x$。

$$Zn + H_2SO_4 = ZnSO_4 + H_2\uparrow$$

$$65 \quad\quad\quad\quad\quad\quad\quad\quad\quad 2$$

$$x \quad\quad\quad\quad\quad\quad\quad\quad\quad 0.4\text{ g}$$

$$\frac{65}{x} = \frac{2}{0.4\text{ g}}$$

$$x = \frac{65 \times 0.4\text{ g}}{2} = 13\text{ g}$$

**答**：需要消耗锌粒 13 g。

---

**第5题（计算题）**

某工厂用赤铁矿（主要成分为 $Fe_2O_3$）炼铁，反应的化学方程式为：

$$Fe_2O_3 + 3CO \xrightarrow{\text{高温}} 2Fe + 3CO_2$$

若该赤铁矿中 $Fe_2O_3$ 的质量分数为 80%，要炼出 560 kg 铁，需要赤铁矿多少千克？

（相对原子质量：$Fe = 56$，$O = 16$，$C = 12$）

——2023年河南省中考（改编）

**答案**：

$Fe_2O_3$ 的相对分子质量 $= 56 \times 2 + 16 \times 3 = 160$

$2Fe$ 的相对质量 $= 56 \times 2 = 112$

设需要 $Fe_2O_3$ 的质量为 $y$。

$$Fe_2O_3 + 3CO \xrightarrow{\text{高温}} 2Fe + 3CO_2$$

$$160 \quad\quad\quad\quad\quad\quad\quad\quad 112$$

$$y \quad\quad\quad\quad\quad\quad\quad\quad 560\text{ kg}$$

$$\frac{160}{y} = \frac{112}{560\text{ kg}}$$

$$y = \frac{160 \times 560\text{ kg}}{112} = 800\text{ kg}$$

设需要赤铁矿的质量为 $z$：

$$z \times 80\% = 800\text{ kg}$$

$$z = \frac{800\text{ kg}}{80\%} = 1000\text{ kg}$$

**答**：需要赤铁矿 1000 kg。

## 五、本课小结

```
化学方程式
    │
    ├── 定义：用化学式表示化学反应的式子
    │
    ├── 意义
    │       ├── 宏观：反应物和生成物的种类
    │       ├── 微观：各物质分子（原子）的个数比
    │       └── 量的：各物质质量比（计量数 × 相对分子质量）
    │
    ├── 书写步骤（"写配注等"）
    │       ├── 写：写出反应物和生成物的化学式
    │       ├── 配：配平（系数最简整数比）
    │       │       ├── 最小公倍数法（通用）
    │       │       └── 观察法（简单反应）
    │       ├── 注：标注反应条件（点燃/加热/催化剂）
    │       │       和状态符号（↑气体 / ↓沉淀）
    │       └── 等：用"==="表示方程式成立
    │
    └── 化学方程式计算（"设写找列算答"）
            ├── 设：设未知量 x
            ├── 写：写出配平方程式
            ├── 找：找出质量比（计量数×相对分子质量）
            ├── 列：列比例方程
            ├── 算：解出 x
            └── 答：写出答案（含单位）
```

**关键规则速记**：

| 规则 | 内容 |
|-----|------|
| 配平原则 | 只改系数，不改化学式 |
| "↑"使用 | 生成物是气体 + 反应物无气体 |
| "↓"使用 | 生成物是沉淀 + 反应在溶液中 |
| 计算关键 | 代入的必须是纯物质质量 |
| 质量守恒 | 计算结果可用质量守恒验证 |

**学习建议**：化学方程式的书写和计算是初三化学的核心技能，贯穿整个初三学习。建议预习时重点理解配平方法，多练习书写常见方程式，打好基础。
