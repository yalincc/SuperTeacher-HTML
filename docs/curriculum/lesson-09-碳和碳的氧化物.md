# 第9课：碳和碳的氧化物

## 一、学习目标

- 了解碳的几种单质（金刚石、石墨、C₆₀）的结构特点和主要用途
- 掌握碳的化学性质（可燃性、还原性），能正确书写相关化学方程式
- 掌握二氧化碳的物理性质和化学性质，理解CO₂与水、石灰水的反应
- 掌握二氧化碳的实验室制取方法和一氧化碳的性质，能鉴别CO和CO₂并进行相互转化

## 二、知识点梳理

### 核心概念

碳元素是自然界中非常重要的一种元素，它可以形成多种单质和化合物。本课围绕碳的单质、碳的化学性质，以及两种碳的氧化物（CO₂和CO）展开学习。

**1. 碳的单质**

碳元素可以形成多种单质，主要原因在于碳原子之间的排列方式不同。

| 单质 | 结构特点 | 主要物理性质 | 主要用途 |
|------|----------|------------|---------|
| 金刚石 | 正八面体结构，每个碳原子与4个碳原子以共价键相连 | 无色透明、正八面体晶体，天然最硬的物质 | 切割玻璃、切割大理石、做钻头、装饰品（钻石） |
| 石墨 | 层状结构，每层碳原子排列成平面正六边形 | 深灰色、有金属光泽、质软、有滑腻感、能导电 | 铅笔芯、电极、润滑剂 |
| C₆₀ | 由60个碳原子构成的分子，形似足球 | 分子晶体 | 超导材料、新材料研究 |

> **关键理解**：金刚石和石墨的物理性质差异巨大，是因为**碳原子的排列方式不同**。这就好比同样的砖头，不同的砌法可以建成完全不同的建筑。虽然物理性质不同，但它们都由碳元素组成，化学性质相似。

**2. 无定形碳**

木炭、活性炭、焦炭、炭黑等都是无定形碳，它们主要是由石墨的微小晶体和少量杂质构成的。

- **活性炭**：有很强的**吸附性**（物理变化），可用于净水、防毒面具、制糖脱色等
- **木炭**：也有吸附性，可用于取暖、冶炼金属
- **焦炭**：用于冶炼生铁
- **炭黑**：用于制墨汁、油漆、橡胶添加剂

> ⚠️ 注意："碳"是元素名称，"炭"是具体物质名称。说"碳元素""碳酸""二氧化碳"用"碳"，说"木炭""活性炭""炭黑"用"炭"。

### 重点内容（★标注）

**★ 碳的化学性质**

（1）**可燃性**

碳在氧气中充分燃烧：
$$C + O_2 \xrightarrow{\text{点燃}} CO_2$$

碳在氧气中不充分燃烧：
$$2C + O_2 \xrightarrow{\text{点燃}} 2CO$$

> 碳燃烧是放热反应，碳是重要燃料。燃烧产物取决于氧气的量：氧气充足生成CO₂，氧气不充足生成CO。

（2）**还原性**（★重点）

碳能在高温下夺取某些金属氧化物中的氧，使金属氧化物被还原。

碳还原氧化铜：
$$2CuO + C \xrightarrow{\text{高温}} 2Cu + CO_2\uparrow$$

现象：黑色粉末逐渐变为红色（生成铜），产生能使澄清石灰水变浑浊的气体（CO₂）。

碳还原氧化铁：
$$2Fe_2O_3 + 3C \xrightarrow{\text{高温}} 4Fe + 3CO_2\uparrow$$

> 碳的还原性在冶金工业中有重要应用，用于冶炼金属。

（3）**稳定性**

常温下碳的化学性质不活泼，受日光照射或与空气、水分接触都不容易变化。因此古代字画用碳素墨水书写可以保存很久。

**★★★ 二氧化碳的性质**

**物理性质：**
- 无色无味的气体
- 密度比空气大（标准状况下，CO₂密度约1.977 g/L，空气约1.293 g/L）
- 能溶于水（1体积水约溶解1体积CO₂）
- 固态CO₂叫"干冰"，升华时吸收大量热

> 干冰升华吸热，可用于人工降雨、冷藏食品、舞台烟雾效果。

**化学性质：**

① 一般不燃烧，也不支持燃烧

> 这是CO₂灭火的原因。但注意：镁条能在CO₂中燃烧（2Mg + CO₂ → 2MgO + C），所以CO₂不能用于扑灭镁等活泼金属的火灾。

② 与水反应生成碳酸

$$CO_2 + H_2O = H_2CO_3$$

碳酸能使紫色石蕊溶液变红。

> 实验设计：将干燥的紫色石蕊纸花放入CO₂中不变色，润湿的紫色石蕊纸花放入CO₂中变红，说明是CO₂与水反应生成的碳酸使石蕊变色，而不是CO₂本身。

碳酸不稳定，容易分解：
$$H_2CO_3 = H_2O + CO_2\uparrow$$

> 加热变红的石蕊溶液，红色会褪去恢复紫色，就是因为碳酸受热分解。

③ 与石灰水反应（★检验CO₂的标准方法）

$$CO_2 + Ca(OH)_2 = CaCO_3\downarrow + H_2O$$

现象：澄清石灰水变浑浊。这是**检验CO₂的常用方法**。

**★ 二氧化碳的实验室制取**

**药品：** 大理石（或石灰石）与稀盐酸

$$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$$

**实验装置要点：**
- **发生装置**：固液常温型（与制取H₂相同），不能用加热装置
- **收集装置**：向上排空气法（CO₂密度比空气大，能溶于水不能用排水法）

**注意事项：**
① 不能用稀硫酸代替稀盐酸：因为CaSO₄微溶于水，会覆盖在大理石表面阻止反应继续进行
② 不能用浓盐酸：浓盐酸有强挥发性，挥发出的HCl气体会混入CO₂中使气体不纯
③ 不能用碳酸钠代替大理石：Na₂CO₃是粉末状，与盐酸反应速率太快，不便于收集

**★ 一氧化碳的性质**

**物理性质：** 无色无味的气体，密度比空气略小，难溶于水

**化学性质：**

① **可燃性**

$$2CO + O_2 \xrightarrow{\text{点燃}} 2CO_2$$

现象：蓝色火焰。点燃前必须验纯（与H₂一样）。

② **还原性**（★重点）

$$CuO + CO \xrightarrow{\Delta} Cu + CO_2$$

$$Fe_2O_3 + 3CO \xrightarrow{\text{高温}} 2Fe + 3CO_2$$

> CO的还原性是高炉炼铁的原理。

③ **毒性**

CO能与血液中的血红蛋白结合，使其丧失携氧能力，造成人体组织缺氧。

> 冬天用煤炉取暖要防止煤气中毒，就是防止CO中毒。CO无色无味，难以察觉，特别危险。

**CO和CO₂的鉴别与转化**

**鉴别方法：**

| 方法 | CO | CO₂ |
|------|----|----|
| 通入澄清石灰水 | 无变化 | 变浑浊 |
| 通入紫色石蕊溶液 | 无变化 | 变红 |
| 点燃 | 能燃烧，蓝色火焰 | 不燃烧 |

**相互转化：**

CO → CO₂：CO燃烧或CO与CuO等反应
$$2CO + O_2 \xrightarrow{\text{点燃}} 2CO_2$$

CO₂ → CO：高温下CO₂与碳反应
$$CO_2 + C \xrightarrow{\text{高温}} 2CO$$

### 难点突破

**难点1：金刚石和石墨物理性质差异的原因**

很多同学难以理解"同种元素组成的不同单质，物理性质可以截然不同"。关键在于：**物质的性质取决于结构**。金刚石中碳原子形成空间网状结构，每个碳原子被牢固地"锁"在四个共价键中，所以极其坚硬；石墨中碳原子形成层状结构，层与层之间作用力很弱，容易滑动，所以质软有滑腻感。

> 类比：同样是一堆积木，搭成实心方块和搭成扁平的层，其牢固程度自然不同。

**难点2：CO还原金属氧化物的实验操作**

CO还原CuO的实验中，必须注意：
- 实验开始时先通CO再加热（排尽装置内空气，防止CO与空气混合加热爆炸）
- 实验结束时先停止加热，继续通CO至冷却（防止生成的铜在高温下被空气中的氧气氧化）
- 尾气必须处理（通入石灰水或点燃），防止CO污染空气

> 口诀："CO早出晚归，酒精灯迟到早退"

**难点3：CO₂与NaOH和Ca(OH)₂反应的区别**

CO₂通入NaOH溶液无明显现象（因为生成的Na₂CO₃溶于水），但反应确实发生了：
$$2NaOH + CO_2 = Na_2CO_3 + H_2O$$

CO₂通入Ca(OH)₂溶液有白色沉淀（CaCO₃不溶于水），有明显现象：
$$Ca(OH)_2 + CO_2 = CaCO_3\downarrow + H_2O$$

> 因此检验CO₂用石灰水，而吸收CO₂常用NaOH溶液（吸收更充分）。

## 三、典型例题

**例题1：** 某同学设计了如下实验：将CO₂分别通入①干燥的紫色石蕊纸花、②喷了水的紫色石蕊纸花、③喷了醋酸的紫色石蕊纸花中。请回答：

（1）纸花①不变色，说明什么？
（2）纸花②变红，说明什么？
（3）纸花③变红，与纸花②变红的原因是否相同？

**解析：**

（1）纸花①不变色，说明CO₂本身不能使石蕊变色，排除了CO₂直接使石蕊变色的可能。

（2）纸花②变红，说明CO₂与水反应生成了碳酸（H₂CO₃），碳酸使石蕊变红。这里起作用的是H₂CO₃，不是CO₂本身。

（3）不相同。纸花②变红是因为CO₂与水反应生成H₂CO₃使石蕊变色；纸花③变红是因为醋酸（CH₃COOH）直接使石蕊变色。两者虽然都是酸使石蕊变色，但酸的来源不同。

**例题2：** 实验室制取CO₂时，某同学用以下药品组合进行实验，请判断能否使用并说明理由：

（1）大理石 + 稀硫酸
（2）石灰石 + 浓盐酸
（3）碳酸钠 + 稀盐酸
（4）大理石 + 稀盐酸

**解析：**

（1）不能用。稀硫酸与大理石反应生成微溶的CaSO₄，覆盖在大理石表面，阻碍反应继续进行，反应会很快停止。

（2）不能用。浓盐酸有强挥发性，挥发出的HCl气体混入CO₂中，使制得的气体不纯。

（3）不能用。碳酸钠是粉末状，与稀盐酸反应速率过快，不易控制，不便于收集气体。

（4）能用。大理石是块状固体，与稀盐酸反应速率适中，便于控制，产生的CO₂较纯。这是实验室制取CO₂的标准药品组合。

### 参考答案与解析

**1. B**

A错误，金刚石和石墨都由碳元素组成，是同种元素的不同单质。B正确，碳原子排列方式不同是导致物理性质差异的根本原因。C错误，C₆₀是由碳元素组成的单质，不是化合物。D错误，活性炭使红墨水褪色是吸附作用，属于物理变化。

**2. C**

C错误。CO₂本身不能使石蕊变色，是CO₂与水反应生成的碳酸（H₂CO₃）使紫色石蕊变红。A、B、D均正确。

**3. C**

A错误，稀硫酸与大理石反应生成的CaSO₄微溶，覆盖在大理石表面使反应停止。B错误，应先通CO排尽空气再加热，防止爆炸。C正确，这是检验CO₂的标准方法。D错误，CO₂能溶于水，不能用排水法收集。

**4.** C + O₂ $\xrightarrow{\text{点燃}}$ CO₂ ； 2C + O₂ $\xrightarrow{\text{点燃}}$ 2CO ； CO ； CO与血液中的血红蛋白结合，使血红蛋白丧失携氧能力，造成人体组织缺氧

**5.**

（1）固液常温型发生装置；向上排空气法收集装置；因为该反应是固体与液体在常温下反应，且CO₂密度比空气大、能溶于水故用向上排空气法收集。

（2）将燃着的木条放在集气瓶口，若木条熄灭，说明已收集满。

（3）Fe₂O₃ + 3CO $\xrightarrow{\text{高温}}$ 2Fe + 3CO₂ ； CO（CO夺取了Fe₂O₃中的氧，是还原剂）

## 五、本课小结

```
碳和碳的氧化物
├── 碳的单质
│   ├── 金刚石——最硬——切割、钻头
│   ├── 石墨——质软导电——铅笔芯、电极
│   └── C₆₀——分子结构——新材料
│   ※ 物理性质不同→碳原子排列方式不同
│
├── 碳的化学性质
│   ├── 可燃性：C+O₂→CO₂（充足）/ 2C+O₂→2CO（不足）
│   ├── 还原性：2CuO+C→2Cu+CO₂↑
│   └── 常温稳定性
│
├── 二氧化碳（CO₂）★★★
│   ├── 物理性质：无色无味、密度>空气、溶于水、干冰升华吸热
│   ├── 化学性质：不燃烧不支持燃烧
│   │            与水→H₂CO₃（石蕊变红）
│   │            与Ca(OH)₂→CaCO₃↓（检验CO₂）
│   └── 实验室制取：CaCO₃+2HCl→CaCl₂+H₂O+CO₂↑
│       （大理石+稀盐酸，向上排空气法收集）
│
└── 一氧化碳（CO）★
    ├── 可燃性：2CO+O₂→2CO₂（蓝色火焰）
    ├── 还原性：CuO+CO→Cu+CO₂（炼铁原理）
    ├── 毒性：与血红蛋白结合
    └── 鉴别：石灰水/石蕊溶液/点燃法
```
