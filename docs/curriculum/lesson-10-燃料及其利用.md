# 第10课：燃料及其利用

## 一、学习目标

- 掌握燃烧的三个条件，理解三者缺一不可的关系
- 掌握灭火的原理和方法，能分析常见灭火实例
- 了解化石燃料（煤、石油、天然气）的组成和综合利用方式
- 理解燃料燃烧对环境的影响，认识新能源开发的意义
- 形成九年级化学上册整体知识框架，掌握预习阶段复习方法

## 二、知识点梳理

### 核心概念

燃料的利用与人类生活密切相关。本课从燃烧的本质出发，学习燃烧的条件和灭火的原理，再扩展到常见燃料及其对环境的影响，最后展望新能源的开发利用。

### 重点内容（★标注）

**★★★ 燃烧的条件**

燃烧需要同时满足三个条件，缺一不可：

| 条件 | 说明 | 举例 |
|------|------|------|
| ① 有可燃物 | 能够燃烧的物质 | 纸张、木柴、酒精等 |
| ② 与氧气（或空气）接触 | 可燃物必须与助燃物接触 | 燃烧需要空气 |
| ③ 温度达到着火点 | 使可燃物着火燃烧的最低温度 | 纸的着火点约130℃，白磷着火点40℃ |

> **着火点**是物质本身的属性，一般情况下不可改变。不同物质的着火点不同。

**经典实验：燃烧条件的探究**

实验设计：在500 mL烧杯中加入300 mL热水，放入一小块白磷；在烧杯上盖一片薄铜片，铜片上一端放一小块白磷，另一端放一小块红磷。

| 实验现象 | 分析 |
|---------|------|
| 铜片上的白磷燃烧 | 满足三个条件：可燃物、与空气接触、热水加热达到着火点 |
| 铜片上的红磷不燃烧 | 未达到着火点（红磷着火点约240℃） |
| 热水中的白磷不燃烧 | 未与氧气接触 |

> 补充：向热水中通入氧气后，水中的白磷也能燃烧，进一步证明三个条件缺一不可。

**★ 灭火的原理和方法**

灭火的原理就是破坏燃烧的三个条件之一，使其至少有一个条件不满足：

| 灭火原理 | 对应方法 | 实例 |
|---------|---------|------|
| 隔离可燃物 | 将可燃物与火源隔离 | 森林火灾中设置隔离带 |
| 隔绝氧气 | 使可燃物与氧气隔绝 | 用湿布盖住着火的油锅、CO₂灭火器 |
| 降温至着火点以下 | 将温度降到着火点以下 | 用水灭火（水蒸发吸热） |

> ⚠️ 常见误区：灭火不是"降低着火点"。着火点是物质固有的属性，一般不能改变。灭火是"使温度降到着火点以下"。

**常见灭火器：**

| 灭火器类型 | 灭火原理 | 适用范围 |
|-----------|---------|---------|
| 泡沫灭火器 | 喷出大量CO₂和泡沫覆盖在燃烧物表面，隔绝空气 | 木材、棉布等一般火灾 |
| 干粉灭火器 | 利用压缩的CO₂吹出干粉，干粉受热分解产生CO₂和覆盖层 | 一般火灾及油、气火灾 |
| CO₂灭火器 | 喷出液态CO₂，降温和隔绝氧气 | 图书、精密仪器火灾 |

> 注意：电器着火先断电再灭火；油锅着火不能用水（油比水轻，会浮在水面上继续燃烧）。

**易燃易爆物的安全知识**

- 可燃性气体（H₂、CO、CH₄等）点燃前必须验纯
- 面粉厂、加油站等场所严禁烟火（可燃性粉尘和气体遇明火可能爆炸）
- 爆炸极限：可燃性气体或粉尘与空气混合在一定浓度范围内遇明火会爆炸

> 加油站闻到的"油气味"就是汽油蒸气，所以严禁拨打手机、严禁烟火。

**★ 化石燃料**

化石燃料是古代生物遗骸经长期地质作用形成的，是**不可再生能源**。

**1. 煤——"工业的粮食"**

煤是复杂的混合物，主要含碳元素，还含少量氢、氮、硫、氧等元素。

**煤的综合利用（煤的干馏/煤的焦化）：**

将煤隔绝空气加强热，使煤分解为：

| 产物 | 用途 |
|------|------|
| 焦炭 | 冶炼金属 |
| 煤焦油 | 化工原料（提取多种化合物） |
| 焦炉气 | 燃料 |

> 煤的干馏是**化学变化**（煤分解生成了新物质）。

**2. 石油——"工业的血液"**

石油是复杂的混合物，主要含碳、氢元素。

**石油的分馏：**

利用石油中各成分的**沸点不同**，将石油分离成不同产品。

| 产物 | 用途 |
|------|------|
| 液化石油气 | 燃料 |
| 汽油 | 汽车燃料 |
| 航空煤油 | 飞机燃料 |
| 柴油 | 柴油机燃料 |
| 润滑油 | 润滑 |
| 石蜡 | 蜡烛等 |
| 沥青 | 铺路 |

> 石油的分馏是**物理变化**（利用沸点不同分离，没有生成新物质）。

**3. 天然气**

天然气的主要成分是**甲烷（CH₄）**。

甲烷燃烧：
$$CH_4 + 2O_2 \xrightarrow{点燃} CO_2 + 2H_2O$$

> 甲烷是最简单的有机物。点燃甲烷前必须验纯（甲烷与空气混合遇明火可能爆炸）。

**甲烷的检验**：点燃后在火焰上方罩一个干冷的烧杯，烧杯内壁出现水雾；迅速倒转烧杯，注入少量澄清石灰水振荡，石灰水变浑浊。证明燃烧产物有H₂O和CO₂，说明可燃气体含氢元素和碳元素。

> 沼气的主要成分也是甲烷。农村利用秸秆、粪便发酵产生沼气，既解决了环境污染，又提供了清洁能源。

**★ 燃料燃烧对环境的影响**

**1. 酸雨**

化石燃料（特别是煤）燃烧会排放SO₂和NO₂等气体，这些气体溶于雨水形成酸雨。

酸雨的危害：
- 腐蚀建筑物和雕塑
- 使土壤酸化，影响农作物生长
- 污染水体，影响水生生物
- 腐蚀金属制品

> 煤中含硫元素，燃烧时生成SO₂：S + O₂ → SO₂

**2. 温室效应**

大量燃烧化石燃料排放CO₂，导致大气中CO₂浓度升高，加剧温室效应。

温室效应的影响：
- 全球气温升高
- 冰川融化，海平面上升
- 气候异常

> CO₂是主要的温室气体，此外还有CH₄、O₃、氟利昂等。

**3. 其他污染**

汽车尾气中含有CO、NO₂、含铅化合物等污染物。

**新能源的开发和利用**

| 新能源 | 优点 | 应用 |
|-------|------|------|
| 氢能 | 燃烧产物是水，无污染；热值高 | 火箭燃料、氢燃料电池 |
| 太阳能 | 清洁无污染，取之不尽 | 太阳能发电、太阳能热水器 |
| 风能 | 清洁无污染 | 风力发电 |
| 地热能 | 清洁 | 地热发电、供暖 |
| 核能 | 能量巨大 | 核电站 |
| 乙醇（C₂H₅OH） | 可再生（由粮食发酵制取） | 乙醇汽油 |

氢气作为理想燃料的优点：
① 来源广（可由水电解制取）
② 热值高（等质量H₂燃烧放出的热量约为汽油的3倍）
③ 产物无污染（生成水）

但目前氢能未广泛使用的原因：制取成本高、储存和运输困难。

乙醇燃烧：
$$C_2H_5OH + 3O_2 \xrightarrow{点燃} 2CO_2 + 3H_2O$$

> 乙醇俗称酒精，可由高粱、玉米等发酵制取，属于可再生能源。

### 难点突破

**难点1：燃烧三个条件的综合应用**

很多同学单独理解三个条件没问题，但遇到综合题目就出错。关键在于：

- 控制变量法分析：改变一个条件，其他条件不变
- 白磷和红磷的对比实验是核心：同在铜片上（都有空气），白磷着火点低能燃烧，红磷着火点高不燃烧→说明着火点是关键因素
- 白磷在热水中不燃烧（无氧气）→通入氧气后燃烧→说明氧气是关键因素

**难点2：煤的干馏与石油的分馏的区别**

这是容易混淆的概念：

| | 煤的干馏 | 石油的分馏 |
|--|---------|-----------|
| 变化类型 | 化学变化 | 物理变化 |
| 原理 | 隔绝空气加强热，煤分解为新物质 | 利用沸点不同，分离混合物 |
| 产物 | 焦炭、煤焦油、焦炉气 | 汽油、柴油等不同馏分 |

> 记忆口诀："干馏是化学变化（干化），分馏是物理变化（分物）"

**难点3：选择灭火方法的判断**

根据火灾类型选择合适灭火方法：
- 电器着火：先断电，不能用泡沫灭火器（水能导电）
- 油锅着火：盖锅盖，不能用水（油比水轻）
- 图书档案着火：用CO₂灭火器（不留痕迹）
- 森林火灾：设隔离带（隔离可燃物）

## 三、典型例题

**例题1：** 下列灭火方法中，分别利用了什么灭火原理？

（1）森林着火时，在火场前方砍出一条隔离带
（2）炒菜时油锅着火，盖上锅盖
（3）用扇子扇灭蜡烛

**解析：**

（1）**隔离可燃物**。砍出隔离带后，火没有可燃物可以蔓延，从而达到灭火的目的。

（2）**隔绝氧气**。盖上锅盖后，锅内的氧气被消耗殆尽且无法补充，火焰因缺氧而熄灭。

（3）**使温度降到着火点以下**。扇子扇风加速空气流动带走热量，使蜡烛的温度降到着火点以下而熄灭。

> 注意区分：用扇子扇炉火越扇越旺（提供更多氧气），而扇蜡烛却灭了（带走热量降温）。同样的操作，因为对象不同，效果截然不同。

**例题2：** 某同学用如图装置探究可燃物燃烧的条件，请回答：

A. 铜片上放白磷，烧杯中放白磷，热水温度约80℃
B. 另一组用红磷替代铜片上的白磷

（1）铜片上的白磷能否燃烧？为什么？
（2）烧杯中的白磷能否燃烧？为什么？
（3）如何使烧杯中的白磷燃烧？
（4）用红磷替代铜片上白磷后，红磷能否燃烧？为什么？

**解析：**

（1）能燃烧。铜片上的白磷同时满足三个条件：是可燃物、与空气中的氧气接触、热水加热使温度达到了白磷的着火点（40℃）。

（2）不能燃烧。虽然白磷是可燃物、温度也达到了着火点，但没有与氧气接触（浸没在水中）。

（3）向烧杯中通入氧气（或空气），使水中白磷与氧气接触，三个条件同时满足即可燃烧。

（4）不能燃烧。红磷的着火点约为240℃，80℃的热水不能使红磷达到着火点，虽然红磷是可燃物且与空气接触，但不满足"温度达到着火点"这一条件。

### 参考答案与解析

**1. C**

A错误，爆炸不一定由燃烧引起（如锅炉爆炸、轮胎爆炸等也可能是物理爆炸）。B错误，室内起火打开门窗会提供更多氧气，使火势更大，应关闭门窗。C正确，盖锅盖隔绝了空气（氧气），达到灭火目的。D错误，燃烧需要三个条件同时满足，仅温度达到着火点还不够，还需要可燃物和氧气。

**2. C**

A错误，化石燃料是不可再生能源，用一点少一点。B错误，煤的干馏是化学变化，煤在隔绝空气条件下分解生成新物质。C正确，石油的分馏利用各成分沸点不同进行分离，没有新物质生成，是物理变化。D错误，天然气的主要成分是甲烷（CH₄），不是一氧化碳。

**3. C**

A有利于保护环境，减少化石燃料使用。B有利于减少汽车尾气污染。C不利于保护环境，大量使用化石燃料会排放更多SO₂、NO₂、CO₂等污染物，加剧酸雨和温室效应。D有利于保护环境，植物可以吸收CO₂。

**4.** 有可燃物；与氧气（或空气）接触；温度达到着火点；隔离可燃物；CO₂不能燃烧也不支持燃烧且密度比空气大，覆盖在燃烧物表面隔绝氧气，同时液态CO₂汽化吸热降温

$$**5.** 2H₂ + O₂ \xrightarrow{点燃} 2H₂O；制取成本高，储存和运输困难；C₂H₅OH + 3O₂ \xrightarrow{点燃} 2CO₂ + 3H₂O；太阳能、氢能、风能、地热能、潮汐能、核能等（任写两种）$$

```
燃料及其利用
├── 燃烧与灭火
│   ├── 燃烧三条件★★★：可燃物 + 氧气 + 达到着火点（缺一不可）
│   ├── 灭火三原理★：隔离可燃物 / 隔绝氧气 / 降温至着火点以下

│
├── 化石燃料★
│   ├── 煤——干馏（化学变化）→焦炭、煤焦油、焦炉气
│   ├── 石油——分馏（物理变化）→汽油、柴油、沥青等

│
├── 环境影响★
│   ├── 酸雨←SO₂、NO₂
│   └── 温室效应←CO₂
│
└── 新能源
    ├── 氢能（理想燃料：来源广、热值高、无污染）
    ├── 太阳能、风能、地热能、核能
    └── 乙醇（C₂H₅OH，可再生能源）
```

### 复习建议

1. **以化学方程式为核心**：上册涉及的化学方程式约20个，要能准确默写，注明反应条件（点燃、加热、高温、催化剂等）和符号（↑、↓）

2. **抓典型实验**：空气中氧气含量的测定、电解水、CO₂的制取和性质、燃烧条件的探究——这四个实验是上册重点，要理解原理、装置、操作和现象

3. **理清概念关系**：
   - 物理变化 vs 化学变化（是否有新物质生成）
   - 物理性质 vs 化学性质（是否通过化学变化表现）
   - 单质 vs 化合物（是否由同种元素组成）
   - 纯净物 vs 混合物（是否由一种物质组成）

4. **重视计算基础**：相对分子质量计算、元素质量分数计算、化学方程式计算——这些是中考必考内容

5. **联系生活实际**：化学来源于生活，灭火、环保、能源、材料等知识都与生活紧密相关，理解起来会更深刻

> 预习阶段的重点不在于做题，而在于**理解概念、掌握原理**。建议每课后用5-10分钟回顾本课小结中的知识框架，在脑海中形成清晰的知识网络。开学后跟随老师深入学习时，你会发现预习阶段的理解会让新知识更容易吸收！
