# 第6课：水与化合价

## 一、学习目标

- 掌握电解水实验的原理和结论，理解水的组成
- 了解氢气的性质和验纯方法
- 区分单质和化合物的概念
- 掌握水的净化方法（沉淀、过滤、吸附、蒸馏）
- 学会鉴别硬水和软水
- 理解化合价的概念，熟记常见元素化合价
- 能利用化合价书写化学式并进行相关计算

## 二、知识点梳理

### 核心概念

**1. 水的组成——电解水实验**

实验装置：在水中加入少量稀硫酸（或氢氧化钠）增强导电性，通直流电。

实验现象：
- 两个电极上均有气泡产生
- 与电源负极相连的电极产生的气体多，与电源正极相连的电极产生的气体少
- 体积比：负极:正极 = 2:1

气体检验：
- 负极气体：能燃烧，产生淡蓝色火焰 → **氢气（H₂）**
- 正极气体：能使带火星的木条复燃 → **氧气（O₂）**

实验结论：
- 水由**氢元素**和**氧元素**组成
- 水的化学式为 H₂O（由实验数据推算：氢气与氧气体积比为2:1，即水分子中氢原子与氧原子个数比为2:1）

化学方程式：2H₂O ════ 2H₂↑ + O₂↑（通电）

> 口诀记忆："正氧负氢，氢二氧一"——正极产生氧气，负极产生氢气；氢气体积是氧气的2倍。

**2. 氢气的性质和验纯**

物理性质：无色无味气体，密度比空气小（最轻的气体），难溶于水。

化学性质：
- **可燃性**：2H₂ + O₂ ════ 2H₂O（点燃）——产生淡蓝色火焰
- **还原性**：H₂ + CuO ════ Cu + H₂O（加热）——黑色粉末变红

验纯方法：
1. 收集一试管氢气，用拇指堵住管口
2. 管口朝下，移近火焰，移开拇指点火
3. 若发出**尖锐的爆鸣声**→ 不纯，需重新收集
4. 若发出**轻微的"噗"声**→ 纯净，可以安全点燃

> 安全提醒：点燃氢气前**必须验纯**！否则可能发生爆炸。

**3. 单质和化合物**

| 类别 | 定义 | 举例 | 特点 |
|------|------|------|------|
| 单质 | 由同种元素组成的纯净物 | O₂、Fe、C、H₂ | 只含一种元素 |
| 化合物 | 由不同种元素组成的纯净物 | H₂O、CO₂、NaCl、CaCO₃ | 含两种或以上元素 |
| 氧化物 | 由两种元素组成，其中一种是氧元素的化合物 | H₂O、CO₂、Fe₂O₃ | 化合物的子集 |

> 注意：
> - 单质和化合物都属于**纯净物**
> - 氧化物一定是化合物，化合物不一定是氧化物
> - 含氧元素的化合物不一定是氧化物（如 KMnO₄ 含氧但含三种元素，不是氧化物）

### 重点内容（★标注）

**★ 水的净化：沉淀、过滤、吸附、蒸馏**

| 方法 | 原理 | 作用 | 去除对象 |
|------|------|------|----------|
| 沉淀 | 利用重力使不溶性杂质沉降 | 使水变澄清 | 大颗粒不溶性杂质 |
| 过滤 | 把液体与不溶于液体的固体分离 | 除去不溶性杂质 | 不溶性固体杂质 |
| 吸附 | 活性炭有吸附性 | 除去色素和异味 | 色素、异味物质 |
| 蒸馏 | 液体沸腾汽化后冷凝 | 得到纯净水 | 几乎所有杂质 |

净化程度：沉淀 < 过滤 < 吸附 < 蒸馏

> 过滤操作要点：**一贴二低三靠**
> - 一贴：滤纸紧贴漏斗内壁
> - 二低：滤纸边缘低于漏斗边缘；液面低于滤纸边缘
> - 三靠：烧杯口紧靠玻璃棒；玻璃棒下端紧靠三层滤纸处；漏斗下端紧靠烧杯内壁

**硬水和软水的鉴别**

- **硬水**：含有较多可溶性钙、镁化合物的水
- **软水**：不含或含较少可溶性钙、镁化合物的水

鉴别方法：取样，加入**肥皂水**搅拌
- 泡沫多、浮渣少 → 软水
- 泡沫少、浮渣多 → 硬水

硬水软化的方法：
- 生活中：**煮沸**（将可溶性钙镁化合物转化为不溶性沉淀）
- 实验室/工业：**蒸馏**

**★ 化合价的概念和常见元素化合价**

化合价是用来表示元素在化合物中原子个数比的数值。

核心规则：
1. 化合物中，各元素化合价的代数和 = **0**
2. 单质中，元素的化合价 = **0**
3. 金属元素在化合物中通常显正价
4. 非金属元素与金属化合时显负价，与氧化合时显正价
5. 氧通常显 -2 价，氢通常显 +1 价

常见元素化合价（必背）：

| 元素 | 化合价 | 元素 | 化合价 | 元素 | 化合价 |
|------|--------|------|--------|------|--------|
| H | +1 | O | -2 | C | +2,+4 |
| Na | +1 | K | +1 | Ca | +2 |
| Mg | +2 | Al | +3 | Fe | +2,+3 |
| Cu | +1,+2 | Zn | +2 | Ag | +1 |
| Ba | +2 | Mn | +2,+4,+7 | S | -2,+4,+6 |
| N | -3,+3,+5 | Cl | -1,+1,+5,+7 | P | +3,+5 |

常见原子团化合价：

| 原子团 | 化合价 |
|--------|--------|
| 氢氧根 OH | -1 |
| 硝酸根 NO₃ | -1 |
| 硫酸根 SO₄ | -2 |
| 碳酸根 CO₃ | -2 |
| 铵根 NH₄ | +1 |
| 磷酸根 PO₄ | -3 |

> 化合价记忆口诀（选背）：
> 一价钾钠氯氢银，二价氧钙钡镁锌，三价铝铁四价碳，铜汞二价最常见。

**★ 利用化合价书写化学式**

方法步骤（交叉法/约简交叉法）：

1. 写出组成化合物的元素符号（正价左，负价右）
2. 标出各元素的化合价
3. 将化合价的绝对值交叉，作为对方元素的原子个数
4. 约简为最简整数比
5. 检验：各元素化合价 × 原子个数 的代数和 = 0

例1：写氧化铝的化学式
1. Al O（铝正价在左，氧负价在右）
2. Al⁺³ O⁻²
3. 交叉：Al₂O₃
4. 检验：(+3)×2 + (-2)×3 = 6-6 = 0 ✅

例2：写二氧化碳的化学式
1. C O
2. C⁺⁴ O⁻²
3. 交叉：C₂O₄
4. 约简：CO₂
5. 检验：(+4)×1 + (-2)×2 = 4-4 = 0 ✅

**有关化学式的计算**

1. **相对分子质量** = 各原子的相对原子质量 × 原子个数之和
   - 例：H₂O 的相对分子质量 = 1×2 + 16×1 = 18

2. **元素质量比** = 各元素的相对原子质量 × 原子个数之比
   - 例：H₂O 中 m(H):m(O) = (1×2):(16×1) = 2:16 = 1:8

3. **元素质量分数** = (某元素的相对原子质量 × 原子个数 / 相对分子质量) × 100%
   - 例：H₂O 中 H 的质量分数 = (1×2/18) × 100% = 11.1%

4. **物质中某元素的质量** = 物质质量 × 该元素质量分数

### 难点突破

**难点1：纯净物与混合物 vs 单质与化合物**

很多同学容易把这两组分类搞混，关键是要分清分类标准：

```
物质 ┬─ 纯净物（一种物质） ┬─ 单质（一种元素）：O₂、Fe
     │                    └─ 化合物（多种元素） ┬─ 氧化物（两种元素含氧）
     │                                         └─ 其他化合物
     └─ 混合物（多种物质）：空气、矿泉水、合金
```

- 纯净物/混合物：按**物质种类**分
- 单质/化合物：按**元素种类**分（都是纯净物下的分类）

**难点2：化合价与化学式的互相推求**

已知化学式求化合价：
- 原则：化合物中化合价代数和 = 0
- 例：求 KMnO₄ 中 Mn 的化合价
  - K 为 +1 价，O 为 -2 价
  - 设 Mn 为 x：(1) + x + (-2)×4 = 0 → x = +7
  - 所以 KMnO₄ 中 Mn 为 +7 价

**难点3：同一元素在不同化合物中化合价可能不同**

- FeO 中 Fe 为 +2 价（Fe²⁺，亚铁）
- Fe₂O₃ 中 Fe 为 +3 价（Fe³⁺，铁）
- 区分方法：高价读"铁"，低价读"亚铁"
  - Fe₂O₃ → 氧化铁（三氧化二铁）
  - FeO → 氧化亚铁

## 三、典型例题

**例题1**：下列物质中，属于氧化物的是（ ）

A. O₂  B. KMnO₄  C. H₂O  D. Ca(OH)₂

**解析**：

氧化物的判断标准：①由两种元素组成 ②其中一种是氧元素

- A：O₂ 是单质（只含一种元素），不是氧化物
- B：KMnO₄ 含 K、Mn、O 三种元素，不是氧化物
- C：H₂O 含 H、O 两种元素，其中一种是氧元素，是氧化物 ✅
- D：Ca(OH)₂ 含 Ca、O、H 三种元素，不是氧化物

**答案：C**

---

**例题2**：已知某化合物的化学式为 R₂O₃，其中氧元素的质量分数为 30%，求：
（1）R 的相对原子质量；
（2）R 的化合价。

**解析**：

（1）设 R 的相对原子质量为 x

R₂O₃ 的相对分子质量 = 2x + 16×3 = 2x + 48

氧元素质量分数 = (16×3)/(2x + 48) × 100% = 30%

48/(2x + 48) = 0.3

48 = 0.3(2x + 48)

48 = 0.6x + 14.4

0.6x = 33.6

x = 56（R 为铁 Fe）

（2）R₂O₃ 中，O 为 -2 价，设 R 为 y 价：

2y + (-2)×3 = 0 → 2y = 6 → y = +3

**答案：R 的相对原子质量为56（铁元素），R 的化合价为+3价。**

（2）硝酸铵中氮、氢、氧三种元素的质量比；

——2023年江西中考

---

### 参考答案与解析

**1. 答案：B**

解析：电解水实验中，正极产生氧气，负极产生氢气。口诀"正氧负氢"。与电源正极相连的电极上产生的气体是氧气。

**2. 答案：D**

解析：四种净水方法中，沉淀只能去除大颗粒不溶性杂质；过滤去除不溶性固体杂质；吸附去除色素和异味；蒸馏可将水沸腾汽化后冷凝，得到几乎不含任何杂质的纯净水，净化程度最高。

**3. 答案：B**

解析：利用化合物中化合价代数和为0计算：
- A：FeO 中 Fe 为 +2 价
- B：Fe₂O₃ 中 Fe 为 +3 价
- C：Fe₃O₄ 中 Fe 有 +2 和 +3 两种价态（可看作 FeO·Fe₂O₃）
- D：FeCl₂ 中 Fe 为 +2 价

所以 Fe₂O₃ 中铁元素化合价最高（+3），选B。

**4. 答案：**

（1）氢气；O₂

（2）2:1

（3）氢元素和氧元素

（4）2H₂O ════ 2H₂↑ + O₂↑（通电）

解析：电解水实验中，负极（A管）产生氢气，正极（B管）产生氧气；氢气与氧气的体积比约为2:1；实验说明水由氢元素和氧元素组成；电解水的化学方程式注意配平和条件。

**5. 答案：**

（1）NH₄NO₃ 的相对分子质量 = 14 + 1×4 + 14 + 16×3 = 14 + 4 + 14 + 48 = 80

（2）m(N):m(H):m(O) = (14×2):(1×4):(16×3) = 28:4:48 = 7:1:12

（3）氮元素质量分数 = (14×2/80) × 100% = 28/80 × 100% = 35.0%

（4）40g 硝酸铵中含氮元素质量 = 40g × 35.0% = 14g

解析：注意 NH₄NO₃ 中含有2个N原子，计算氮元素相关量时不要遗漏。元素质量比要化为最简整数比。质量分数保留一位小数。
