# v1.3 UI 改造 — 技术方案

> 状态：**✅ 已实施** | 创建：2026-06-14 | 完成：2026-06-15
>
> 改动详情见 `version.md` v1.3 条目

---

## 设计决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 动画方案 | CSS `@keyframes` + Tailwind `@theme` | 零依赖，全平台 |
| 配色方案 | 实验橙（#e8600c）+ 暖色背景 | 教育产品，温暖专业 |
| 变量管理 | Tailwind v4 `@theme`（`src/index.css`） | 一次定义，全局 token |
| 暗色模式 | 不做 | v1.3 不引入，后续可加 |
| 第三方动效库 | 不用 | 零依赖原则 |

## 采用的动画

| 动画 | 用途 | 实现 |
|------|------|------|
| `animate-slide-up` | 卡片入场 | `@keyframes` translateY |
| `animate-fade-in` | 解析展开 | `@keyframes` opacity |
| `transition` | hover/active 状态 | CSS transition |

## 未采用（备忘）

| 方案 | 原因 |
|------|------|
| `@starting-style` | 仅 Chrome 117+ |
| View Transitions API | 仅 Chrome 111+ |
| IntersectionObserver 滚动渐入 | 可做但 v1.3 不做 |
| CSS confetti | 过度，教育产品保持克制 |
| Framer Motion / GSAP | 违反零依赖 |
| 独立 `src/styles/skins/` 文件 | `@theme` 方案更简洁 |
