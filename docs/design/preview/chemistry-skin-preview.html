化学反应速率的定义

化学反应速率通常用单位时间内反应物浓度的减少量， 或生成物浓度的增加量来表示。

v = Δc / Δt

其中：v 表示反应速率，Δc 表示浓度变化量，Δt 表示时间变化量。

【选择题】
下列关于化学反应速率的说法，正确的是？
A
反应速率越大，反应越快
B
温度升高，反应速率减小
C
催化剂能改变反应速率
D
反应物浓度越大，速率越小
场景：提示框 Callout
⚠️
易错提醒
催化剂在化学反应前后质量和化学性质不变，但物理性质（如状态）可能改变。不要笼统地说"性质不变"。
💡
解题技巧
判断化学反应速率快慢时，注意区分"反应速率大"和"反应快"——前者是定量描述，后者是定性判断，二者含义不同。
ℹ️
补充说明
反应速率的单位通常为 mol/(L·s) 或 mol/(L·min)，具体取决于反应进行的快慢程度。
📖
记忆口诀
温浓压催加快跑，降温减压慢悠悠。固体纯液不参与，速率只论气溶液。

影响化学反应速率的因素

下表总结了影响化学反应速率的主要因素及其规律：

因素 变化方向 对速率的影响 原理
温度 升高 ↑ 加快 ↑ 活化分子比例增大
浓度 增大 ↑ 加快 ↑ 单位体积活化分子数增多
压强 增大 ↑ 加快 ↑（仅气体） 等效于增大浓度
催化剂 加入 加快 ↑ 降低活化能

化学反应速率的要点

掌握以下要点，理解反应速率的核心概念：

  • 反应速率是平均速率，不是瞬时速率
  • 同一反应中，不同物质表示的速率之比 = 化学计量数之比
  • 固体和纯液体的浓度视为常数，不用于表示反应速率
  • 反应速率通常指正值，无论用反应物还是生成物表示

实验步骤（有序列表）：

  1. 准备等浓度的盐酸两份，分别标记为 A、B
  2. A 组加热至 60°C，B 组保持室温
  3. 同时加入等质量的锌粒
  4. 记录收集 50mL 氢气所需的时间
  5. 比较两组数据，得出结论
场景：选择题 — 答错反馈
【选择题】
下列有关催化剂的说法，正确的是？
A
催化剂在反应前后性质不变
B
催化剂能加快所有反应的速率
C
催化剂在反应前后质量和化学性质不变
D
没有催化剂反应就不能进行
❌ 回答错误，正确答案是 C
催化剂在化学反应前后，质量和化学性质不变，但物理性质（如状态、形状）可能改变。A 选项说"性质不变"过于笼统，包含了物理性质，因此不正确。催化剂具有选择性，不能加快所有反应（B 错）。某些反应不需要催化剂也能进行（D 错）。
场景：选择题 — 答对反馈
【选择题】
增大反应物浓度，化学反应速率加快的原因是？
A
活化能降低
B
单位体积内活化分子数增多
C
分子总数减少
D
反应放出更多热量
✅ 回答正确！
增大反应物浓度，使单位体积内的分子总数增多，其中活化分子数也随之增多（活化分子比例不变）。有效碰撞次数增加，因此反应速率加快。A 选项是催化剂的作用原理，C、D 与浓度无关。
✓ 回答正确！
