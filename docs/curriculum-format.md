# MD 教案编写规范

---

## 一、文件结构

```markdown
# 第N课：课时标题

## 一、学习目标      ← 目标列表
## 二、知识点梳理    ← 内容块（段落/表格/提示框/公式/列表）
## 三、典型例题      ← 每题一个 ### 标题
## 四、课后练习      ← 按题型分组
## 五、本课小结      ← 树状总结
```

五个模块顺序固定，不可调整。

---

## 二、各模块格式

### 学习目标

```markdown
## 一、学习目标
- 了解xxx
- ★掌握xxx        ← ★ 标记重点，会高亮
```

### 知识点梳理

用 `### 三级标题` 分小节，支持以下内容块：

**表格：**
```markdown
| 列1 | 列2 |
|-----|-----|
| 数据 | 数据 |
```

**提示框：**
```markdown
> ⚠️ 易错提醒：内容...    ← 红色
> 💡 解题技巧：内容...    ← 绿色
> 📖 记忆口诀：内容...    ← 紫色
> 📌 补充说明：内容...    ← 蓝色
```

**公式（两种）：**

| 类型 | 语法 | 用途 |
|------|------|------|
| 独立块 `$$...$$` | `$$2H_2 + O_2 \xrightarrow{点燃} 2H_2O$$` | 化学方程式（必须用此格式） |
| 内联 `$...$` | `空气主要含 $N_2$ 和 $O_2$` | 简短化学式、符号 |

**公式规则：**

| 规则 | ✅ 正确 | ❌ 错误 |
|------|---------|---------|
| 方程式用 `$$` 块 | `$$C + O_2 \xrightarrow{点燃} CO_2$$` | `$C + O_2 \xrightarrow{点燃} CO_2$` |
| 条件直接写中文 | `\xrightarrow{点燃}` | `\xrightarrow{\text{点燃}}` |
| 标签放公式外 | `方程式：` + `$$...$$` | `$$方程式：...$$` |
| 单位可用 `\text{}` | `14\text{ g}` | — |

**列表：**
```markdown
- 无序列表
1. 有序列表
```

### 典型例题

```markdown
### 例1：题目名称
题干内容（  ）
A. 选项A
B. 选项B

**答案**：B
**解析**：解析内容...
**来源**：2024XX中考    ← 可选
```

### 课后练习

```markdown
### 选择题
1. 题干（  ）
   A. 选项A
   B. 选项B
   **答案**：B
   **解析**：解析...

### 判断题
2. 判断题题干。
   **答案**：对
   **解析**：解析...

### 填空题
3. 内容___，另一个空___。
   **答案**：答案1, 答案2
   **备选**：答案1别名|答案1, 答案2别名    ← 可选容错
   **解析**：解析...

### 简答题
4. 简答题题目。
   **参考答案**：答案内容...
   **得分要点**：要点1；要点2    ← 可选
```

### 本课小结

```markdown
## 五、本课小结
- 知识点1
  - 子知识点1
  - 子知识点2
```

用 `- ` + 缩进（2空格/级），最多3层。

## 三、前端渲染规范

> 以下定义前端组件**实际渲染**的标记。MD 中写的标记必须在此列表内，否则显示为原始文本。

### 3.1 renderInline 支持的标记

`renderInline()` 函数处理以下行内标记（适用于段落、题干、选项、解析、列表项等所有文本字段）：

| 标记 | 语法 | 渲染效果 | 示例 |
|------|------|----------|------|
| 行内公式 | `$...$` | KaTeX 渲染 | `$O_2$` → O₂（下标） |
| 加粗 | `**...**` | `<strong>` | `**重点**` → **重点** |
| 斜体 | `*...*` | `<em>` | `*强调*` → *强调* |

**注意**：`$...$` 内联公式依赖 KaTeX。若 LaTeX 语法错误，KaTeX 会显示原始文本（不报错）。

### 3.2 独立内容块渲染

| 块类型 | 组件 | 说明 |
|--------|------|------|
| `$$...$$` 公式块 | EquationBlock | KaTeX 渲染独立公式 |
| 表格 | TableBlock | Markdown 表格 → HTML 表格 |
| 提示框 `>` | CalloutBlock | 支持内联渲染（加粗/公式） |
| 列表 `-` / `1.` | ListBlock | 支持内联渲染（加加粗/公式） |
| 段落（其他文本） | ParagraphBlock | 支持内联渲染 |

### 3.3 支持渲染的字段

| 字段 | 所属组件 | 支持 renderInline |
|------|---------|:-----------------:|
| 段落 content | ParagraphBlock | ✅ |
| 提示框 title / content | CalloutBlock | ✅ |
| 列表项 text | ListBlock | ✅ |
| 选择题 stem | ChoiceExercise | ✅ |
| 选择题 option.text | ChoiceExercise | ✅ |
| 判断题 stem | TrueFalseExercise | ✅ |
| 填空题 segments | FillExercise | ✅ |
| 简答题 question | ShortAnswerExercise | ✅ |
| 简答题 referenceAnswer | ShortAnswerExercise | ✅ |
| 解析 analysis | AnswerReveal | ✅ |

### 3.4 不支持的标记

| 标记 | 原因 |
|------|------|
| 代码块 ` ``` ` | 前端无渲染组件 |
| 标题 `#` / `##` / `###` | 仅在 MD 结构中使用，不作为内容块渲染 |
| 链接 `[text](url)` | 未实现 |
| 图片 `![alt](url)` | 未实现 |
