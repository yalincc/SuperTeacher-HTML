# MD教案编写规范

> 本规范以 lesson-01 为标准模板，所有新教案必须严格遵循此格式。

---

## 一、文件结构（五个模块固定顺序）

每课必须包含以下五个二级标题模块，**顺序不可调整**：

```markdown
# 第N课：课时标题

## 一、学习目标
（目标列表）

## 二、知识点梳理
（知识点内容，可包含多个三级标题小节）

## 三、典型例题
（例题，每题一个三级标题）

## 四、课后练习
（练习题，按题型分组）

## 五、本课小结
（树状知识点总结）
```

---

## 二、各模块详细规范

### 2.1 文件标题

```markdown
# 第1课：走进化学世界
```

格式：`# 第N课：课时标题`

---

### 2.2 学习目标

```markdown
## 一、学习目标
- 了解化学的定义和研究对象
- ★理解物理变化和化学变化的概念及本质区别
- ★掌握化学变化的判断方法
- 了解物理性质和化学性质的区别
- 认识化学在生活中的应用
```

**规则：**
- 每行一个目标，以 `- ` 开头
- 重点目标前加 `★`（会高亮显示）
- 目标用动词开头：了解、理解、掌握、认识、学会等

---

### 2.3 知识点梳理

```markdown
## 二、知识点梳理

### 小节标题1
内容段落...

### 小节标题2
内容段落...
```

**规则：**
- 每个知识点用 `### 三级标题` 分隔
- 支持以下内容块类型：

#### 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 数据 | 数据 | 数据 |
```

#### 提示框

```markdown
> ⚠️ 易错提醒：提示内容...

> 💡 解题技巧：技巧内容...

> 📖 记忆口诀：口诀内容...

> 📌 补充说明：说明内容...
```

**提示框类型（按关键词自动识别）：**
| 关键词 | 类型 | 颜色 |
|--------|------|------|
| ⚠️ 易错提醒 | warning | 红色 |
| 💡 解题技巧 | tip | 绿色 |
| 📖 记忆口诀 | mnemonic | 紫色 |
| 📌 其他 | note | 蓝色 |

#### 公式

行内公式：`$公式$`，如 `$CO_2$`

独立公式：`$$公式$$`，如：
```markdown
$$2H_2 + O_2 \xrightarrow{点燃} 2H_2O$$
```

#### 列表

无序列表：
```markdown
- 项目1
- 项目2
```

有序列表：
```markdown
1. 步骤1
2. 步骤2
```

### 2.4 典型例题

```markdown
## 三、典型例题

### 例1：判断化学变化
下列变化中，属于化学变化的是（  ）
A. 冰雪融化
B. 铁钉生锈
C. 酒精挥发
D. 矿石粉碎

**答案**：B
**解析**：化学变化的本质特征是有新物质生成。
- A. 冰雪融化：水从固态变为液态，没有新物质生成 → 物理变化
- B. 铁钉生锈：铁与氧气、水反应生成铁锈（主要成分Fe₂O₃），有新物质 → 化学变化

**来源**：2024北京中考
```

**规则：**
- 每个例题用 `### 例N：题目名` 开头
- 选择题选项：`A. 选项内容` 每行一个
- 必须包含 `**答案**：X`
- 必须包含 `**解析**：解析内容`
- 可选 `**来源**：出处`
- 解析内容支持列表、公式等

### 2.5 课后练习

```markdown
## 四、课后练习

### 选择题
1. 题干内容（  ）
   A. 选项A
   B. 选项B
   C. 选项C
   D. 选项D

   **答案**：B
   **解析**：解析内容...
   **来源**：2024广州中考

### 判断题
2. 判断题题干内容。

   **答案**：对
   **解析**：解析内容...

### 填空题
3. 化学变化与物理变化的本质区别是___。铁生锈属于___变化。

   **答案**：是否有新物质生成, 化学
   **备选**：有没有新物质生成|有无新物质生成, 化学
   **解析**：解析内容...

### 简答题
4. 简答题题目内容。

   **参考答案**：参考答案内容...
   **得分要点**：要点1；要点2；要点3
```

**规则：**

#### 选择题
- 格式：`N. 题干（  ）`
- 选项：`A. 选项` 每行一个，缩进3个空格
- 答案：单个字母，如 `**答案**：B`

#### 判断题
- 格式：`N. 判断题题干。`
- 答案：`对` 或 `错`

#### 填空题
- 填空位置用 `___`（三个下划线）表示
- 多个空用逗号分隔答案
- 可选 `**备选**` 提供容错答案，用 `|` 分隔同义答案，逗号分隔不同空

#### 简答题
- 答案用 `**参考答案**：`
- 可选 `**得分要点**` 用分号分隔

### 2.6 本课小结

```markdown
## 五、本课小结
- 化学变化与物理变化
  - 本质区别：是否有新物质生成
  - 化学变化的常见现象
    - 发光发热
    - 颜色变化
    - 气体生成
    - 沉淀生成
  - 注意：现象≠本质，要结合"新物质"判断
- 物理性质与化学性质
  - 物理性质：无需化学变化即可表现
  - 化学性质：在化学变化中表现
```

**规则：**
- 用 `- ` 开头，通过缩进表示层级（2个空格一级）
- 支持最多3层嵌套
- 内容简洁，突出要点

```bash
npm run convert
```
