# v1.9 规划：化学课程按教材 18 课题重新生成

> 日期：2026-06-18 | 状态：已定稿

## 背景

2024 人教版九年级化学教材有 7 单元 18 课题。当前课程只有 10 课且标题/单元与教材不一致。为方便学生对照课本学习，按教材目录重新生成全部 18 课。

## 新课程结构

```
第一单元 走进化学世界
  01 物质的变化和性质
  02 化学实验与科学探究

第二单元 空气和氧气
  03 我们周围的空气
  04 氧气
  05 制取氧气

第三单元 物质构成的奥秘
  06 分子和原子
  07 原子结构
  08 元素

第四单元 自然界的水
  09 水资源及其利用
  10 水的组成
  11 物质组成的表示

第五单元 化学反应的定量关系
  12 质量守恒定律
  13 化学方程式

第六单元 碳和碳的氧化物
  14 碳单质的多样性
  15 碳的氧化物
  16 二氧化碳的实验室制取

第七单元 能源的合理利用与开发
  17 燃料的燃烧
  18 化石能源的合理利用
```

## 实施步骤

1. 更新 `src/data/courses/chemistry/course.json` — 18 课题结构
2. 清空现有 lesson JSON
3. 用 generate-lesson skill 逐课生成 18 个 lesson-XX.json + exercises
4. `npm run validate` + `npm run build` 验证
