# v1.7 规划：知识点页面"互动教科书"改造

> 日期：2026-06-17 | 状态：讨论中（未定稿）

---

## 背景

v1.7 已上线的基础功能（Hero 头部、阅读进度条、Sticky CTA、SectionObjectives 任务卡、SectionExamples 翻卡式自测）验证通过，但整体页面观感**像最初级博客**——字体排版普通，缺乏教学产品特有的视觉语言。

---

## 当前问题诊断

| 问题 | 具体表现 |
|------|---------|
| 排版层级扁平 | 段落只有 `text-sm` + `leading-relaxed`，没有阅读节奏感 |
| 布局单调 | 全页纵向堆叠，所有区块等宽，没有视觉引导线 |
| 知识面板像后台 | 手风琴折叠是 Admin 面板常见样式，不像学习产品 |
| 内容块缺乏打磨 | 段落是 `<p>`、表格是普通 table、列表是 `<ul>`，无教学场景视觉语言 |
| 区块之间无关联感 | 目标→知识→例题→小结是断裂的，缺少"学习旅程"的感觉 |

---

## 改进方向（待讨论确认）

### 方向一：排版升级

- 段落增大行高（`leading-loose`）+ 首行缩进 + 适当字间距
- 重要概念用**高亮背景色块**（类似教科书黄色荧光笔效果），而不只是加粗
- 表格增加左侧彩色边条 + hover 行高亮 + 单元格内联公式渲染

### 方向二：布局改造

- 知识面板从"手风琴折叠"改为**全展开卡片流**，每张知识卡有左侧色带 + 编号标签
- 区块之间加**过渡分隔符**（渐变横线 + 图标），形成学习旅程的节奏感
- 目标区从列表改为**横向标签流**（chip/tag 形式），重点目标用大号标签

### 方向三：内容块增强

- **段落块**：关键术语自动高亮（`$...$` 语法渲染为带背景的 chip）
- **表格块**：首列固定宽度 + 斑马纹加对比度 + 圆角卡片包裹
- **提示框块**：从 `border-l-4` 细线改为**全包裹卡片**，带图标圆形背景 + 标题行
- **公式块**：增大内边距 + 暖黄背景卡 + 编号标签（如 "方程 1.1"）
- **列表块**：自定义 bullet（橙色圆点/数字徽章）+ 缩进线

### 方向四：交互微动效

- 知识点卡片进入视口时**交错淡入**（stagger animation）
- 阅读进度条改为**彩色渐变**（橙→绿），暗示学习完成度
- 例题区增加"**思考中...**"按钮状态，点击后 3 秒倒计时再显示答案（防止偷看）
