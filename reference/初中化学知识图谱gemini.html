基础核心

一、微观奥秘：化合价与化学式

1. 常考重点

  • 化合价原则：在化合物中，正负化合价的代数和为 零；单质中元素的化合价为 零。
  • 根据化合价写化学式：正价在前，负价在后，标价交叉，约简定数。

2. 避坑指南

【易错点】有些元素在不同的物质中会显示不同的化合价。例如在 FeCl2（氯化亚铁）中铁元素为 +2价，在 FeCl3（氯化铁）中铁元素为 +3价。中考命名听到“亚”字，铁指 +2 价，铜指 +1 应特别注意。
重难点引发点

二、化学灵魂：质量守恒定律与方程式

1. 质量守恒定律的“微观本质”

  • 六个不变：宏观（物质总质量、元素种类、元素质量）+ 微观（原子种类、原子数目、原子质量）。
  • 两个一定变：宏观（物质种类）+ 微观（分子种类）。
  • 一个可能变：分子总数目可能改变。

2. 方程式计算的三个踩分点

① 步骤规范：设、写、找、列、解、答，缺一不可。
② 纯量代入：化学方程式计算中，代入的数据必须是纯净物的质量。若有杂质或气体，必须先进行质量换算。
高频考点

三、金属大乱斗：活动性顺序及其应用

1. 金属活动性顺序表

K Ca Na Mg Al, Zn Fe Sn Pb (H), Cu Hg Ag Pt Au

2. 置换反应的两大应用重难点

  • 金属与酸反应：只有排在 (H) 前面的金属才能置换出酸中的氢（浓硫酸、硝酸除外）。
  • 金属与盐溶液反应：“前置后，盐可溶”。即前面的金属能把后面的金属从其可溶性盐溶液中置换出来。
  • 滤渣与滤液分析（中考压轴选择题）：向混合盐溶液中加入一种金属，遵循“弱者先被置换”的原则。例如向 Cu(NO3)2 和 AgNO3 的混合溶液中加锌，锌会优先与 AgNO3 反应。

四、大压轴：酸碱盐的性质与复分解反应

1. 常见酸碱盐的化学性质

物质分类 核心化学性质（高频反应）
酸 (H+) 酸 + 活泼金属 → 盐 + H2  |  酸 + 金属氧化物 → 盐 + H2O
碱 (OH-) 碱 + 非金属氧化物 → 盐 + H2O  |  碱 + 酸 → 盐 + H2O (中和反应)
盐 盐 + 金属 → 新盐 + 新金属  |  盐 + 酸/碱 → 新盐 + 新酸/新碱

2. 复分解反应发生的条件（死记硬非）

  • 反应物条件：有酸可不溶，无酸皆可溶（即反应物中如果有酸，另一种物质不溶也可以反应；如果全是盐或碱反应，则反应物必须都能溶于水）。
  • 生成物条件：必须有 沉淀(↓)、气体(↑) 或 水(H2O) 至少一种生成。

五、提分关键：气体制取与物质推断

1. 常见气体的发生与收集装置选择

  • 发生装置：根据反应物的状态和反应条件选择。
    • 固+固加热型（如高锰酸钾制氧气）
    • 固+液常温型（如大理石与稀盐酸制 CO2）
  • 收集装置：根据气体的溶解性和密度选择。
    • 排水法：气体不易溶于水、不与水反应（收集到的气体较纯）。
    • 排空气法：密度比空气大用向上排空气法（如 CO2）；密度比空气小用向下排空气法。

2. 物质推断题的常见“题眼”（突破口）

• 特征颜色：蓝色沉淀 → Cu(OH)2；红褐色沉淀 → Fe(OH)3；黑色固体 → CuO, Fe3O4, 炭粉, 铁粉。
• 特征现象：能使带火星木条复燃的是 O2；能使澄清石灰水变浑浊的是 CO2。
