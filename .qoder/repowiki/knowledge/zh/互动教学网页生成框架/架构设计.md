- **分层架构**：采用内容层（`course.json`, `docs/curriculum/*.md`）与引擎层（React 组件、Hooks）分离的设计，通过构建管道解耦。
- **转换管道**：基于 `scripts/convert-md.mjs` 实现 MD 到结构化 JSON 的预构建转换，按 H2 标题切分模块并解析为 `ContentBlock` 和 `Exercise` 数据模型。
- **注册表模式**：引擎层通过 `ContentBlockRegistry` 和 `ExerciseRegistry` 动态分发渲染组件，支持题型和内容块的模块化扩展。
- **数据流**：静态 JSON 数据（`src/data/lessons/`）由 React Router 按需加载，学习进度通过 `localStorage` 持久化，无全局状态库依赖。
